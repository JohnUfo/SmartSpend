### If you're using .strings files:
1. Open your `Localizable.strings` file for your language (e.g., `ru.lproj/Localizable.strings`)
2. Copy ALL 9 column keys above for your language
3. Paste them into the file
4. Save and rebuild

### If you're using String Catalog (.xcstrings):
1. Open your String Catalog in Xcode
2. Click the + button to add new strings
3. Add each of these 9 keys one by one:
   - `column_expense`
   - `column_amount`
   - `column_category`
   - `column_date`
   - `column_title`
   - `column_or_title_name`
   - `column_title_name_description`
   - `column_amount_price_cost`
   - `column_category_type`
4. For each key, add the translation for your language
5. Save and rebuild

### Verification:
After adding these keys:
1. Clean Build Folder (Cmd+Shift+K)
2. Rebuild the app
3. Go to Settings â†’ Import Data
